# OrgazmDeals Bot 🤖

Официальный бот ручений от @orgazm.

## Установка

```bash
pip install -r requirements.txt
```

Скопируй `.env.example` в `.env` и заполни свои данные:

```bash
cp .env.example .env
```

## Запуск

```bash
python bot.py
```

## Команды админа

| Команда | Описание |
|---|---|
| `/admin` | Панель статистики |
| `/pending` | Все ожидающие заявки |
| `/заявка 5 текст` | Ответить на заявку |
| `/setbanner` | Установить баннер (отправь фото) |
| `/removebanner` | Удалить баннер |

## Несколько админов

В `.env` укажи ID через запятую:
```
ADMIN_ID=174415647,707651857
```
Все админы получают уведомления и могут отвечать на заявки.
